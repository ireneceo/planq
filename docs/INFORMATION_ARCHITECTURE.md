# 06. IA (정보구조) 설계

---

## 1. URL 라우팅 구조

### 인증 전
```
/login                          로그인
/register                       회원가입 (사업자 등록)
/forgot-password                비밀번호 재설정
/invite/:token                  고객 초대 수락 (간편 가입)
```

### Business Owner / Business Member
```
/app                            대시보드 (리다이렉트)
/app/dashboard                  대시보드
/app/talks                      Q Talk — 대화 목록
/app/talks/:conversationId      Q Talk — 대화방
/app/tasks                      Q Task — 할일 목록
/app/tasks/:taskId              Q Task — 할일 상세
/app/files                      Q File — 자료함
/app/files/:clientId            Q File — 고객별 자료
/app/bills                      Q Bill — 청구서 목록
/app/bills/new                  Q Bill — 청구서 작성
/app/bills/:invoiceId           Q Bill — 청구서 상세
/app/notes                      Q Note — 회의/음성 정리
/app/notes/:noteId              Q Note — 결과 상세
/app/clients                    고객 관리
/app/clients/:clientId          고객 상세
/app/team                       팀 관리 (Owner만)
/app/settings                   설정 (Owner만)
/app/settings/profile           사업자 프로필
/app/settings/subscription      구독 관리
/app/settings/notifications     알림 설정
```

### Client (고객)
```
/c/talks                        내 대화
/c/talks/:conversationId        대화방
/c/tasks                        내 할일
/c/files                        공유자료
/c/bills                        청구서
/c/profile                      내 정보
```

### Platform Admin
```
/admin/dashboard                전체 대시보드
/admin/businesses               사업자 관리
/admin/businesses/:id           사업자 상세
/admin/users                    사용자 관리
/admin/audit                    감사 로그
/admin/settings                 플랫폼 설정
```

## 4. 반응형 기준

| 화면 | 너비 | 레이아웃 변화 |
|------|------|-------------|
| Desktop | 1200px+ | Sidebar + Content + Task Panel (3단) |
| Tablet | 768~1199px | Sidebar 접힘 + Content + Task Panel (2단) |
| Mobile | ~767px | 하단 탭바 + Content (1단), Task Panel은 슬라이드 |
