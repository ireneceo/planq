# 아키감사 — 아키텍처 회귀 정기 감사 명령어

버그가 아니라 **구조의 침식**을 잡는 스킬. 큰 사이클 마무리나 월 1회 실행.
산출물은 "현재 상태 → 리스크 → 보완안(직접 vs Irene 승인)" 형식 보고.

---

## Claude 필수 수행 (전부 실측 — 추정 금지)

### 1단계: 가드 3축 전체 실행

```bash
cd /opt/planq/dev-backend && node /opt/planq/scripts/health-check.js
node /opt/planq/scripts/guard-invariants.js --verbose
node /opt/planq/scripts/e2e/run.js --suite mobile,crosscut,l1,tenant
```

### 2단계: 부채 추이 측정 (래칫 베이스라인 대비)

```bash
# 베이스라인 총량 (i18n 하드코딩 / 무스코프 쿼리 / pagination 미적용 / god-file)
node -e "const b=require('/opt/planq/scripts/guards-baseline.json');
for(const k of ['i18n','tenant','pagination','godfile','parity_keys']){
  const v=b[k]||{}; const t=Object.values(v).filter(n=>typeof n==='number').reduce((a,x)=>a+x,0);
  console.log(k, Object.keys(v).length+'개 파일', '총', t);}"
```

- 지난 감사 대비 **부채 총량이 줄었는가**를 보고. 줄었으면 `--update-baseline` 으로 조여 박제.
- 부채가 그대로/늘었으면(래칫은 파일 단위라 이동 가능) 원인 파일 3개 지목.

### 3단계: 구조 실측 (변화분 중심)

```bash
# god-file 상위 (라우트 / 컴포넌트)
wc -l /opt/planq/dev-backend/routes/*.js | sort -rn | head -8
find /opt/planq/dev-frontend/src -name '*.tsx' | xargs wc -l | sort -rn | head -8
# 인증 없는 라우트 파일 (공개 6개 외 신규 발견 = 즉시 보고)
grep -LE "authenticateToken" /opt/planq/dev-backend/routes/*.js
# 응답 헬퍼 이탈 (res.json({ success: 직접 사용)
grep -lE "res\.json\(\{ success:" /opt/planq/dev-backend/routes/*.js
```

기준선 (2026-07-10 실측): 인증 없는 파일 = app_download·blog·platform_public·stripeWebhook·stripeWorkspaceWebhook·internal(내부키) 6개. **이 목록에 새 파일이 나타나면 Fable 게이트 대상.**

### 4단계: 문서 신선도

- `guard-invariants` docfresh 경고 확인 (SYSTEM_ARCHITECTURE/DATABASE_ERD/ONBOARDING/PERMISSION_MATRIX 60일)
- 이번 사이클 변경과 어긋난 문서는 갱신하거나 갱신 필요를 보고

### 5단계: 감사 보고 형식

```
## 아키감사 YYYY-MM-DD
1. 가드 3축: health ✓/✗ · invariants ✓/✗ · e2e ✓/✗
2. 부채 추이: i18n N(±) / tenant N(±) / pagination N(±) / godfile N(±)
3. 신규 리스크: (인증 누락 신규 파일 / 신규 god-file / 헬퍼 이탈 증가 등)
4. 조치: 직접 조인 것(--update-baseline 등) vs Irene 승인 필요(리팩터 후보 우선순위)
```

## 절대 금지

- 이 스킬에서 생명선 코드(access_scope·billing·receiptsDue·task_workflow 전이·auth/security)를 직접 리팩터 — 발견은 보고까지만, 수술은 Irene 승인 + Fable 게이트
- 래칫 베이스라인을 부채가 **늘어난 상태**에서 재기록 (동결 무력화)
- 실측 없는 "구조 건강함" 보고
