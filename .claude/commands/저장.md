# 세션 저장 명령어

현재 Claude Code 대화 내용과 개발 히스토리를 저장합니다.

## 실행 단계

### 1. 현재 세션 상태 저장

`/opt/planq/.claude/session-state.md` 파일을 업데이트합니다:

```markdown
## 현재 작업 상태
**마지막 업데이트:** [현재 날짜시간 UTC]
**작업 상태:** [진행중/완료]

### 진행 중인 작업
- [현재 진행 중인 작업 목록]

### 완료된 작업 (이번 세션)
- [이번 세션에서 완료한 작업들]
- [각 작업에 대한 간단한 설명]
- [수정한 주요 파일 목록]

### 다음 할 일
- [다음에 해야 할 작업들]

### 주요 변경사항
- [중요한 코드 변경 요약]
- [설정 변경 사항]
- [해결한 버그/이슈]
```

### 2. Git 상태 기록

현재 Git 상태도 함께 기록합니다:
- 수정된 파일 목록
- 커밋되지 않은 변경사항
- 최근 커밋 해시

### 3. 저장 완료 알림

저장 완료 후 사용자에게 알립니다:
- 저장된 위치
- 저장된 내용 요약
- 복구 방법 안내

## 복구 방법

새 세션에서 아래 내용을 입력하면 이어서 작업 가능:

```
이전 세션 이어서 작업하고 싶어.
/opt/planq/.claude/session-state.md 읽어줘.
```

## 주의사항

- 이 명령어는 파일에 저장하는 것이므로, 서버가 재시작되어도 유지됩니다
- 민감한 정보(비밀번호, API 키 등)는 저장하지 않습니다
- Git 커밋과는 별개입니다 (코드 변경은 별도로 커밋 필요)
